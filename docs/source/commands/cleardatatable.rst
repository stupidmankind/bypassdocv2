Clear DataTable
===============

Category
--------
DataTable Commands

Description
-----------

This command clear a DataTable. Use this command when you want to clear a new DataTable.

Parameters
----------

**Datatable Name**
	The DataTable object to be cleared of all data.



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------
