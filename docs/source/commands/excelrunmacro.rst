Run Macro
=========

Category
--------
Excel Commands

Description
-----------

This command runs a macro. Use this command when you want to get a run a specific macro in the Excel workbook.

Parameters
----------

**Instance Name**
	Indicate the instance name for the selected workbook

**Macro Name**
	Enter the macro name 

**Parameters**
	Enter the parameter attributes



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\Excel Commands\\Run Macro\\Run Macro.xml`
