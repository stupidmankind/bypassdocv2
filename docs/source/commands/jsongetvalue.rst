Json Get Value
==============

Category
--------
JSon Commands

Description
-----------

This command allows you to extract a data from Json to a variable.

Parameters
----------

**Json String**
	Enter the Json String

**Path to value**
	Please Enter the path to value. Example propery[0].Id

**Result**
	Select the variable to receive the result. 



Returned Value
--------------
	[ReturnValue]

-------------
