Kill Process
============

Category
--------
Programs/Process Commands

Description
-----------

This command allows you to kill a program or a process. Use this command to kill an application by its name such as 'chrome'.

Parameters
----------

**Program Name**
	Select the process name to be killed



Returned Value
--------------
	[ReturnValue]
