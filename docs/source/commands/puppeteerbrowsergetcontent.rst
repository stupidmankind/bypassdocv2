PPT Browser GetContent
======================

Category
--------
Web Browser Commands

Description
-----------

This command allows you to get the whole page of a web browser session.

Parameters
----------

**Instance Name**
	Indicate the instance name for the selected web browser.

**URL**
	Enter the URL to get content.

**Output Text**
	Specify output variable name.



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\Web Browser Commands\\PPT Browser GetContent\\PPT Browser GetContent.xml`
