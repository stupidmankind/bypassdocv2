PPT Browser Get JSON
====================

Category
--------
Web Browser Commands

Description
-----------

This command allows you to get the json from web api of a web browser session.

Parameters
----------

**Instance Name**
	Indicate the instance name for the selected web browser.

**URL**
	Enter the URL to get content.

**Method Type**
	Specify method type.

**Payload (use double quote ONLY)**
	Specify Payload. (use double quote ONLY)

**Output Text**
	Specify output variable name.

**Request Header**
	Please Enter the request header attributes.



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\Web Browser Commands\\PPT Browser Get JSON\\PPT Browser Get JSON.xml`
