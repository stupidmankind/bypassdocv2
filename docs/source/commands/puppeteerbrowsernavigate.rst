PPT Browser Navigate
====================

Category
--------
Web Browser Commands

Description
-----------

This command allows you to navigate a web browser to a URL.

Parameters
----------

**Instance Name**
	Indicate the instance name for the selected web browser.

**URL**
	Enter the URL to navigate to.



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\Web Browser Commands\\PPT Browser Navigate\\PPT Browser Navigate.xml`
