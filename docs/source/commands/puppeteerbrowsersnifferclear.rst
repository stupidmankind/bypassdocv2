PPT Browser Sniffer Clear
=========================

Category
--------
Web Browser Commands

Description
-----------

This command will clean list of request data.

Parameters
----------

**Instance Name**
	Enter the instance name

**Frame**
	Specify Frame Name.

**Sniffer Operation**
	Start/Stop Sniffer



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\Web Browser Commands\\PPT Browser Sniffer Clear\\PPT Browser Sniffer Clear.xml`
