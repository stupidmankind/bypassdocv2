PPT Browser Switch Tab
======================

Category
--------
Web Browser Commands

Description
-----------

This command allows you to switch tab in a web browser session.

Parameters
----------

**Instance Name**
	Enter the instance name

**Browser Property**
	Specify which page property to switch to.

**Value**
	Specify exact value of the Page property to work on.



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\Web Browser Commands\\PPT Browser Switch Tab\\PPT Browser Switch Tab.xml`
