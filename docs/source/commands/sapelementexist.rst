SAP Element Exist
=================

Category
--------
SAP Commands

Description
-----------

This command allows you to check whether the specified element exists.

Parameters
----------

**ID**
	Specify which ID to look at.

**Result**
	Select a variable to receive the result.



Returned Value
--------------
	[ReturnValue]
