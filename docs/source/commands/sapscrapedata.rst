SAP Scrape Data
===============

Category
--------
SAP Commands

Description
-----------

This command allows you to scrape all data in the specific table in a SAP session.

Parameters
----------

**Table ID**
	Specify which table to look at.

**Result**
	Select a variable to receive the result.



Returned Value
--------------
	[ReturnValue]
