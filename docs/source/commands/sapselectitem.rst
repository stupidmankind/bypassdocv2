SAP Select Item
===============

Category
--------
SAP Commands

Description
-----------

This command allows you to select a dropdown item in a SAP session.

Parameters
----------

**Dropdown ID**
	Specify which dropdown to look at.

**Dropdown Item Name**
	Specify which dropdown item to select.



Returned Value
--------------
	[ReturnValue]
