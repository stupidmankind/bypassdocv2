SAP Type
========

Category
--------
SAP Commands

Description
-----------

This command allows you to type into a SAP session.

Parameters
----------

**ID**
	Specify which ID to look at.

**Text**
	Specify Text to type.

**Encryption**
	Indicate if the text in 'TextToSend' is Encrypted.

**GuiTextField**
	Indicate if the Selector ID is GuiTextField



Returned Value
--------------
	[ReturnValue]
