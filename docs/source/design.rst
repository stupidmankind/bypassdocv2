Design
======

**BYpass** is designed for everyone. A BYpass license come with following package.

- **Robot:** The robot that uses to execute RPA scripts.
- **Developer Studio:** A piece of software that use to design RPA scripts. All BYpass package comes with Developer Studio.
- **Control Panel:** A tool that allows to execute RPA scripts remotely.
	

Developer Studio is used to develop RPA scripts, It was designed to work by selecting commands, configure it and execute. There is no coding. The studio allows to open scripts in multiple windows. Check out the topics below for more information.
