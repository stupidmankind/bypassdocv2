Welcome to BYpass's documentation!
===================================

**BYpass** is a RPA software developped by Backyard. BYpass is developed by to make automation easy for everyone. It can use on a simple task to a very complex tasks. Let's get to know the software!

.. note::

   What is RPA?.
   RPA stands for Robotic Process Automation. The role of RPA is to automate repetitive tasks that were previously handled by humans. RPA is the software robot that is programmed to do repetitive tasks across applications and systems.
